GIT (CLI)

기본문법

- gitBash에서 작업 진행

$pwd 현재 나의 위치 확인
$ls 현재 위치에 디렉토리및 파일 확인
$cd 디렉토리 선택
$cd .. 상위 디렉토리 선택
$git init git으로 버전관리 할 프로젝트 선택(git 초기화)
$ls -al 현재 위치에 디렉토리및 파일 확인(숨은 디렉토리까지)
$git add [파일명] "commit" 할 파일 선택
$ git add . "commit" 할 파일 전체 선택
$git commit -m "[메모]" 선택한 파일들을 한 묶음으로 만들고 메모 입력
$git log 생성한 커밋 보기

원격 저장소에 올리기

$git remote add [저장할 주소] $git remote add origin https://github.com/woong2021/project.git
git remote -v 원격저장소 연결상태 확인
$git push origin master gitHub 사이트에 commit 하기

원격 저장소에 받아오기

$git clone [받아올 주소]

$git clone https://github.com/woong2021/project.git .
$rm -rf [디렉토리명] 디렉토리 삭제

LF will be replaced by CRLF "git config --global core.autocrlf true" 입력
