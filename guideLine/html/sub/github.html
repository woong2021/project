GIT (CLI)

CLI란 "Command-line interface"의 약자로 터미널을 통해 상호작용하는 방식

GIT 초기화

git init

CLI 명령어

CLI 명령어
$pwd 현재 나의 위치 확인
$cd 디렉토리 선택
$cd .. 상위 디렉토리 선택
$ls 현재 위치에 디렉토리및 파일 확인
$ls -al 현재 위치에 디렉토리및 파일 확인(숨은 디렉토리까지)

버전 생성

버전 생성
$git add [파일명] "commit" 할 파일 선택
$ git add . "commit" 할 파일 전체 선택
$git commit -m "[메모]" 선택한 파일들을 한 묶음으로 만들고 메모 입력
$git log 생성한 커밋 보기

브렌치

브렌치 생성
$ git checkout -b [브렌치명]
원격 저장소에 브랜치 등록
$ git push origin [브렌치명]
local 브렌치와 remote 브렌치 연결
$ git branch --set-upstream-to origin/[브렌치명]
브렌치 목록 확인
$ git branch

원격 저장소에 올리기

git remote -v 원격저장소 연결상태 확인
$git push origin [브랜치명(master)] gitHub 사이트에 commit 하기

원격 저장소에서 받아오기

$git clone https://github.com/woong2021/project.git .
