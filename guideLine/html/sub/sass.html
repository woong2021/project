  7. 상단에 internal editors 가 선택되어 있는지 확인한 다음 "CSS Editor"를 찿아서 선택한 다음 ok 버튼 클릭

Sass 문법

문법

SASS 문법
주석
  • // Common : CSS로 컴파일 되지 않는 주석
  • /* Common */ : CSS로 컴파일 되는 주석
                                        // *----------------------------
                                        //	Contents
                                        // ----------------------------*
!global
  • 변수의 유효범위를 전역으로 설정
  • (변수를 따로 파일을 만들어 관리하기 때문에 사용안함)
!default
  • 할당되지 않은 변수의 초깃값을 설정
  • 할당되어 있는 변수가 있다면 변수가 기존 할당값을 사용
  • (예로 $color: orange !default; 가 있고 $ color: red; 변수가 후에 생겼다면 $color: red 를 사용)
#{} 문자보간이라 하며 코드의 어디든지 변수 값을 넣을 수 있음

내장함수

내장함수
unquite()
  • 문자에서 따옴표를 제거
  • $family: unquote("woong");
unitless()
  • px등 단위가 있는지 확인
  • if(unitless($height), #{$height}px, $height);
