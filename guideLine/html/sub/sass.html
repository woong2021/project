Eclipse css 편집기와 연결
  1. "window > preferences"로 이동
  2. "General > Editors > File Associations" 로 이동
  3. 파일연결 창에서 오른쪽 상단의 "add..." 버튼을 클릭
  4. File Type 에 "*.scss"를 입력한 다음 ok 버튼을 클릭
  5. *.scss 목록에서 File Associations 항목을 찿아서 선택
  6. *.scss를 선택한 후 하단창 Associated editors에서 "Add"버튼을 클릭
  7. 상단에 internal editors 가 선택되어 있는지 확인한 다음 "CSS Editor"를 찿아서 선택한 다음 ok 버튼 클릭
