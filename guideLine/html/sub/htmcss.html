display: flex

display: flex 이용

2021년 현재 트랜드에 맞춰 레이아웃을 잡을때 display:inline-block를 자제하고 "display: flex"를 지향

display: flex 사용법

세로정렬
  • "align-items: center"를 사용하여 세로 가운데 정렬 가능
가로정렬
  • "justify-content: space-between"를 사용하여 양끝 정렬 가능
아이템 순서
  • "order"를 이용하여 아이템 순서를 지정

TABLE 표기법 설명
<caption>
  • 웹접근성 고려하여 테이블 내에 <caption> 반드시 기재
  • 기재는 하되 레이아웃 구성상 숨김처리 (클래스 "sr-only" 사용)
  • "sr-only" 클래스가 지정 안되었을시 다음 코드 생성
    .sr-only{
            position: absolute;
            width: 1px;
            height: 1px;
            padding: 0;
            margin: -1px;
            overflow: hidden;
            clip: rect(0, 0, 0, 0);
            border: 0;
    }
