시멘틱 태그

다양한 시멘틱태그가 있으나 주로 사용하는 것만 기재

시멘틱 태그에 대한 설명
< header > 이름 그대로 헤더 영역을 의미
< nav > 네비게이션을 의미하며 보통 메뉴 작성때 사용
< main > 컨텐츠가 들어 갈 영역을 의미
< section > 최근 < section > 태그는 사용안하는 추세이나 개인적으로는 필요, 스포츠, 음악, 음식 등 대분류역할
< article >
  • section 이 대분류라면 article 는 중분류나 소분류라 생각하면 됨 (예, 야구, 축구, 배구, 농구)
  • 어디까지나 의미론 적인것이며 article 안에 "section"이 들어갈 수 있음
  • 개인적으로 section(대분류) -> article (중분류) -> section이 아닌 "div (소분류)" 작업되었으면 함
< footer > 이름 그대로 풋터 영역을 의미

image 테그

object-fit

object-fit에 대한 설명
fill 주어진 너비와 높이에 딱 맞도록 사이즈를 조절, 이미지 가로세로 비율 유지되지 않음 (default)
contanin 가로세로 비율을 유지한 채로 사이즈가 조절되지만, 이미지와 컨테이너 간의 비율이 맞지 않는 경우엔 자리가 생김
cover 가로세로 비율을 유지한 채로 사이즈가 조절되어 비율이 맞지 않더라도 이미지를 확대해 컨테이너를 완전히 채움
none 아무것도 하지않고 본래의 이미지 사이즈를 유지
scale-down none 또는 contain 중에 더 적절한 방향으로 이미지 사이즈를 조절

단위에는 여러가지가 있으나 "두가지"만 사용하나 절때 정답은 아닙니다.

단위에 대한 설명
px 기본적인 단위로써 웹에 사용할때 주로 사용(모바일 포함)
vw 모바일 버전에서 "font-size" 크기 지정할때만 사용

display: flex

display: flex 이용

2021년 현재 트랜드에 맞춰 레이아웃을 잡을때 display:inline-block를 자제하고 "display: flex"를 지향

display: flex 사용법

flex 사용법에 대한 설명
세로정렬
  • "align-items: center"를 사용하여 세로 가운데 정렬 가능
가로정렬
  • "justify-content: space-between"를 사용하여 양끝 정렬 가능
아이템 순서
  • "order"를 이용하여 아이템 순서를 지정
