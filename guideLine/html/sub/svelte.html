Svelte

Vue.js로 인한 현재 교육 보류

Svelte 특징

  1. Diffing 이나 Overhead 가 일어나지 않아 처리속도 가 빠르다.

    • diffing : "가상돔(Virtual DOM)"을 생성하고 비교하는 행위
    • Overhead : 어떤 처리를 위해 들어가는, 간접적인 시간이나 메모리
  2. 애플리케이션 상태의 변화가 DOM에 자동으로 반영 (반응성)

Svelte 단점

  • 작은 생태계 (2019년 4월 3릴리즈)
  • runtime으로 동작하지 않기 때문에 CDN 미제공
  • IE 미지원

Svelte 명령어

Svelte 명령어 설명
npx degit sveltejs/template [프로젝트명] 프로젝트 생성및 설치
code . 현재 프로젝트 열기
npm i 프로젝트 모듈 설치
npm run dev 프로젝트 실행
